<p align="center">A static educational website featuring information about the Great Wall of China, with interactive image galleries showcasing the landmark across different seasons, smooth animations, and a warm earth-tone design.</p>

## ✨ Features

- Multi-page navigation with smooth transitions between Home, Gallery, and Contacts
- Animated header with text shadow effects and gradient overlays
- Interactive image gallery displaying the Great Wall in Spring, Fall, and Winter
- Animated background images with pan effects and hover scaling
- Warm, cohesive color scheme inspired by Great Wall landscapes
- Educational content about the Great Wall's history and significance
- Contact section with personal information and learning resources

## 🚀 Getting Started

### ⚙️ Prerequisites

No install required. Runs in any modern browser.

### ⚡ Open locally

1. Open `index.html` in your browser.
2. Use the navigation buttons to explore Home, Gallery, and Contacts sections.
3. Click the play button to animate the main image gallery.
4. Hover over season gallery cards to see zoom effects.

## 🗂️ Files to explore

- `index.html` — page structure with navigation and sections
- `styles/style.css` — visual design, animations, and color scheme
- `script.js` — interactive animations and page transitions
- `images/` — seasonal photos and header background

## 📝 Notes

- The project uses vanilla CSS animations for smooth transitions.
- Color scheme features earth tones: terracotta, olive, cream, and slate.
- Designed as a college assignment with educational and presentational focus.
- All images are included locally for easy offline viewing.

> A polished educational experience celebrating the architectural marvel of the Great Wall.
