<p align="center">A festive static experience built as a holiday assignment, featuring animated snowfall, a seasonal scene, and an audio toggle button.</p>

## ✨ Features

- Animated snowfall effect using JavaScript-driven DOM elements
- Seasonal welcome animation with Santa and holiday visuals
- Play/pause music button for a festive soundtrack
- Simple static frontend design using HTML, CSS, and vanilla JavaScript

## 🚀 Getting Started

### ⚙️ Prerequisites

No install required. Runs in any modern browser.

### ⚡ Open locally

1. Open `index.html` in the browser.
2. Wait for the splash animation to finish.
3. Click the **Music** button to play or pause the holiday audio.

## 🗂️ Files to explore

- `index.html` — page structure and entry point
- `css/style.css` — visual design and animations
- `script.js` — snowfall generation and audio control
- `sounds/christmas sound.mp3` — holiday background music

## 📝 Notes

- The project is a static webpage with no build process.
- Snow is generated dynamically for a lively winter scene.
- Designed as a college assignment with a festive theme.

> A simple seasonal experience for winter greetings and holiday mood.
